# Project Euler Solutions

### C++ Commands
Compile: `g++ x.cpp`  
Run    : `./a.out`  

### Java Commands

Compile: `javac -d bin -cp lib\* *.java`  
Run    : `java -cp bin\;lib\* org.junit.runner.JUnitCore EulerTest`  
Copy   : `cp tmp/pN.java p042.java`  
Clean  : `rm bin/*`
